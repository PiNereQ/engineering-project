# COUPIDYN - Mobile Discount Coupon Exchange App

### Project Description

**COUPIDYN** is a mobile Android application developed as an engineering thesis project. The app enables users to **sell and buy discount coupons**.

The project is built using **Flutter** and follows a client-server architecture.
